# dotfiles

macOS向けのターミナルデザイン設定です．

## Contents

- iTerm2のカラースキーム

## Design

落ち着いた配色と少ない情報量を重視した，ミニマルなターミナル環境を目指しています．

## iTerm2

| 項目 | 設定 |
|---|---|
| Font | JetBrainsMono Nerd Font Mono |
| Font Size | 13 pt |
| Transparency | 25% |
| Blur | 8% |
| Background Image | なし |

### Colors

| 項目 | Hex |
|---|---|
| Background | `#282826` |
| Foreground | `#E6E2D9` |
| Cursor Background | `#D3CFC5` |
| Cursor Foreground | `#282826` |
| Selection Background | `#4B4944` |
| Selection Foreground | `#F1EEE7` |

## Privacy

APIキー，SSH設定，環境変数，ユーザー固有パスなどは管理しません．
